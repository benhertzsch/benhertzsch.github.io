Codes

I am a co-developer of the PhaseSpaceDTFE.jl julia package for the evaluation of multistream density and velocity fields from cosmological N-body simulations. The code is hosted at this repository and registered with the DOI 10.5281/zenodo.16637560. Have a look at the documentation if you’re interested in using the code in your own research, and do not hesitate to reach out to the developers!
