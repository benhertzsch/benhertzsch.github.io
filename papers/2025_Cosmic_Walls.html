Additional Materials – 2025 Cosmic walls paper

This webpage contains additional material for the paper

Benjamin Hertzsch, Job Feldbrugge, Maé Rodriguez, Rien van de Weygaert (2025) “A New Recipe for Pancakes — Caustic Skeleton Constraint Simulations I: On the Reality of Walls in the Cosmic Web”

We provide the following materials accompanying the text:

  • Animations:

    • High-resolution wall centre constraint simulation: fly-through animation (mp4-file) (above) and rotating animation (mp4-file)

    • Formation of a multistream region on a 2D particle mesh: animation (mp4-file) (for source code, contact job.feldbrugge@ed.ac.uk)

    • 3D view of a Zel’dovich pancake: animation (mp4-file) (for source code, contact job.feldbrugge@ed.ac.uk)

  • A3 wall centre constraint fields:

    • 100 Field realisations for the A3 wall centre constraint at (σ, bc) = (2.0, 0.8): folder (zip)

    • 100 Field realisations for the contracting A3 wall centre constraint at (σ, bc) = (2.0, 0.9): folder (zip)

    • 100 Field realisations for (σ, bc) = (1.5, 0.75): folder (zip)

    • 100 Field realisations for (σ, bc) = (1.0, 0.6): folder (zip)

    • 100 Field realisations for (σ, bc) = (2.0, 0.9): folder (zip)

  • ϕ(+ − −) saddle point constraint fields:

    • 100 Field realisations for σ = 2.0: folder (zip)

  • δ(+ + −) saddle point constraint fields:

    • 100 Field realisations for (σ, bc) = (2.0, 0.9): folder (zip)

    • 50 Field realisations for (σ, bc) = (1.5, 0.8): folder (zip)

    • 50 Field realisations for (σ, bc) = (1.0, 0.7): folder (zip)
